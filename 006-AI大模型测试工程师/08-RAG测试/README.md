# 08 RAG 测试

RAG 测试是 AI 大模型测试工程师的重点能力之一。RAG 系统不是只看“模型回答得像不像”，而是要拆开看：资料有没有被正确加载、切分是否合理、检索是否命中、上下文是否相关、模型回答是否忠实于上下文、引用是否准确。

## 本章你要学会什么

- 看懂 RAG 系统链路。
- 区分检索问题和生成问题。
- 设计 RAG 测试集。
- 测试文档切分、召回、排序、答案忠实性和引用。
- 使用人工评估、规则评估和自动化指标评估 RAG。
- 输出一份 RAG 评测报告。

## RAG 测试核心结论

```text
RAG 失败不一定是模型失败。
可能是文档加载失败、切分失败、检索失败、排序失败、Prompt 失败，也可能是模型生成幻觉。
```

所以测试时一定要拆链路。
