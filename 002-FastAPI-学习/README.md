## 推荐学习方式

每个模块建议采用以下节奏：

1. 先看核心概念。
2. 写一个最小可运行示例。
3. 用 Swagger UI 测试接口。
4. 整理笔记。
5. 把知识点合并进一个小项目。

最终目标是完成一个具备用户注册、登录、文章管理、权限控制、数据库持久化和自动化测试的 FastAPI 后端项目。
