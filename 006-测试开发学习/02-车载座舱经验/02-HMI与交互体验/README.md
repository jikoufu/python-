# 02 HMI 与交互体验

HMI 是车载座舱里非常重要的一块。它不只是页面好不好看，而是驾驶场景下用户能不能安全、快速、清楚地完成操作。

本章重点学习：

- HMI 是什么。
- 座舱 HMI 和手机 App 交互有什么区别。
- 页面结构、菜单层级、弹窗、Toast、状态反馈怎么测。
- 夜间模式、多屏联动、触控响应、动效怎么测。
- 如何设计 HMI 测试用例。
- 常见 HMI 问题如何沉淀为经验。
