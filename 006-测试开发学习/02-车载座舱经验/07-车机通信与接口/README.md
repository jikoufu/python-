# 07 车机通信与接口

## 这个模块是什么

智能座舱测试岗位里，**车载总线是辅助技能，不是主战场**。

你要达到的程度是：

- 面试能聊明白原理
- 工作中能看懂报文
- 能和总线工程师正常对话
- 出问题时能配合定位

**不需要**会做总线一致性测试、网络管理开发这类深度内容。

## 学习定位

```text
必须重点投入  →  CAN / CAN FD、车载以太网、DBC 报文解析
了解即可      →  LIN、AUTOSAR、DoIP、TSN
工具入门      →  CANoe（看懂、会基本操作即可）
```

一句话：**CAN 和车载以太网值得多花时间，LIN 和 AUTOSAR 是"了解"级别。**

## 推荐学习方式

1. 先看 `01-车载总线学习计划.md`，了解整体节奏和投入分配
2. 按 02 → 03 → 04 顺序过知识点
3. `05-DBC报文解析实战.md` 一定要动手敲，这是能写进简历的亮点
4. 最后用 `06-面试题汇总.md` 自测，练面试话术

## 重要提醒

- **DBC 读报文是性价比最高的技能**：零成本、面试能展示、工作天天用，还能结合你的 Python 优势
- **没有硬件也能学**：CANoe 有 Demo/仿真模式，cantools 纯软件就能跑
- **别在 LIN 和 AUTOSAR 上花太多时间**：它们是"了解"级别
