# AI 大模型测试工程师学习目录

这个目录用于系统学习 AI 大模型测试工程师需要的能力。它不是单纯的传统测试，也不是只会调用大模型 API，而是把测试工程、Python 自动化、LLM 应用、Prompt、RAG、Agent、评估、安全、性能、成本和平台化能力结合起来。

## 这个岗位要解决什么问题

AI 大模型测试工程师主要负责验证大模型应用是否：

- 功能正确
- 回答质量稳定
- Prompt 修改不引入回归
- RAG 检索和回答可靠
- Agent 工具调用正确
- 不泄露敏感信息
- 不被 Prompt 注入轻易绕过
- 延迟、成本和稳定性可接受
- 能通过自动化测试持续回归

- 设计测试策略
- 编写接口自动化测试
- 编写 AI 回答质量测试
- 编写 Prompt 回归测试
- 编写 RAG 测试集
- 编写 Agent 工具调用测试
- 设计安全测试用例
- 输出评测报告
- 接入 CI 自动回归
- 搭建一个小型 AI 测试框架
