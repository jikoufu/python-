# LangChain 学习路径

## 阶段 1：理解 LangChain 是什么

目标：知道 LangChain 解决什么问题，以及它在 AI 应用架构中的位置。

学习内容：

- LangChain 是什么
- LangChain v1
- LangGraph 是什么
- LangSmith 是什么
- LangChain 和 LlamaIndex 的区别

阶段成果：

- 能解释 LangChain 适合做什么。
- 能区分 LangChain、LangGraph、LangSmith。
- 能知道什么时候用 LangChain，什么时候考虑 LlamaIndex。

## 阶段 2：模型调用与 Prompt

目标：能通过 LangChain 调用大模型，并使用 Prompt 模板组织输入。

学习内容：

- ChatModel
- PromptTemplate
- ChatPromptTemplate
- System Message
- Human Message
- 模型响应对象

建议练习：

1. 安装 LangChain。
2. 配置模型 API Key。
3. 写一个最小模型调用。
4. 写一个 Prompt 模板。
5. 让模型根据变量生成回答。

## 阶段 3：输出解析

目标：让模型输出变成程序容易处理的结构。

学习内容：

- 字符串输出
- JSON 输出
- Pydantic 模型解析
- 格式错误处理
- 输出校验

阶段成果：

- 能让模型输出固定 JSON。
- 能用 Pydantic 校验模型输出。
- 能发现并处理输出格式不稳定的问题。

## 阶段 4：Runnable 与 LCEL

目标：掌握 LangChain 的链式组合方式。

学习内容：

- Runnable
- LCEL
- 管道符 `|`
- `invoke`
- `batch`
- `stream`
- 多步骤组合

核心写法：

```python
chain = prompt | model | parser
result = chain.invoke({"question": "LangChain 是什么？"})
```

阶段成果：

- 能把 Prompt、模型、解析器组合成一条链。
- 能批量处理多个输入。
- 能理解流式输出的基本方式。

## 阶段 5：工具调用与 Agent

目标：让模型不仅能回答，还能调用工具完成任务。

学习内容：

- Tool
- 函数工具
- `create_agent`
- 工具参数
- Agent 执行轨迹
- 工具调用失败处理

建议练习：

1. 写一个天气查询工具。
2. 写一个计算器工具。
3. 创建一个 Agent。
4. 让 Agent 根据用户问题选择工具。
5. 观察 Agent 的输入和输出。

## 阶段 6：记忆与上下文

目标：理解对话历史和长期记忆怎么管理。

学习内容：

- 短期记忆
- 长期记忆
- 对话历史
- 上下文窗口
- 记忆存储
- 上下文压缩

阶段成果：

- 能保存用户对话历史。
- 能在下一轮对话中使用历史上下文。
- 能理解记忆和普通 Prompt 拼接的区别。

## 阶段 7：检索与 RAG

目标：使用 LangChain 构建知识库问答。

学习内容：

- Document
- Loader
- Text Splitter
- Embedding
- Vector Store
- Retriever
- RAG Chain
- Agentic RAG

阶段成果：

- 能加载本地文档。
- 能切分文本并生成向量。
- 能根据问题检索相关内容。
- 能让模型基于检索结果回答。

## 阶段 8：LlamaIndex 对比

目标：知道 LlamaIndex 的强项和适用场景。

学习内容：

- LlamaIndex Document
- Node
- Index
- Query Engine
- Retriever
- 和 LangChain 的区别

阶段成果：

- 能知道 LlamaIndex 更偏数据和索引。
- 能判断项目该选 LangChain、LlamaIndex，还是组合使用。

## 阶段 9：实战项目

目标：完成一个 LangChain 小项目。

推荐项目：

- CLI AI 助手
- 文档问答 RAG
- 带工具调用的查询助手
- AI 测试用例生成器

最终能力：

- 能用 LangChain 调模型。
- 能写 Prompt 模板。
- 能解析结构化输出。
- 能组合 Runnable。
- 能构建简单 Agent。
- 能做基础 RAG。
