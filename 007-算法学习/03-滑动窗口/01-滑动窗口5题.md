# 滑动窗口 5 题

## 学习过程

滑动窗口用于处理“连续子数组 / 连续子串”问题。

基本模板：

```python
left = 0

for right in range(len(nums)):
    # 1. 把 nums[right] 加入窗口

    while 窗口不满足条件:
        # 2. 移除 nums[left]
        left += 1

    # 3. 更新答案
```

核心是明确：

```text
窗口里维护什么？
什么时候扩大？
什么时候收缩？
什么时候更新答案？
```

## 题 1：无重复字符的最长子串

### 题目

给定字符串，找出不含重复字符的最长子串长度。

### 思路

维护一个窗口，窗口内不能有重复字符。

用集合保存窗口内字符。右指针加入字符，如果重复，就不断移动左指针直到不重复。

### Python 代码

```python
def length_of_longest_substring(s: str) -> int:
    seen = set()
    left = 0
    best = 0

    for right, ch in enumerate(s):
        while ch in seen:
            seen.remove(s[left])
            left += 1

        seen.add(ch)
        best = max(best, right - left + 1)

    return best
```

### 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(k)`，`k` 是字符种类数

## 题 2：长度最小的子数组

### 题目

给定正整数数组和目标值 `target`，找出和大于等于 `target` 的最短连续子数组长度。

### 思路

因为数组元素都是正数，所以窗口右扩时和变大，左缩时和变小。

当窗口和满足 `>= target`，尝试缩小窗口。

### Python 代码

```python
from typing import List


def min_sub_array_len(target: int, nums: List[int]) -> int:
    left = 0
    window_sum = 0
    best = float("inf")

    for right, num in enumerate(nums):
        window_sum += num

        while window_sum >= target:
            best = min(best, right - left + 1)
            window_sum -= nums[left]
            left += 1

    return 0 if best == float("inf") else best
```

### 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`

## 题 3：最大连续 1 的个数 III

### 题目

给定二进制数组 `nums` 和整数 `k`，最多可以把 `k` 个 0 变成 1，返回最长连续 1 的长度。

### 思路

窗口内最多允许 `k` 个 0。

右指针扩大窗口，遇到 0 就计数。若 0 的数量超过 `k`，左指针收缩窗口。

### Python 代码

```python
from typing import List


def longest_ones(nums: List[int], k: int) -> int:
    left = 0
    zero_count = 0
    best = 0

    for right, num in enumerate(nums):
        if num == 0:
            zero_count += 1

        while zero_count > k:
            if nums[left] == 0:
                zero_count -= 1
            left += 1

        best = max(best, right - left + 1)

    return best
```

### 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`

## 题 4：字符串的排列

### 题目

给定两个字符串 `s1` 和 `s2`，判断 `s2` 是否包含 `s1` 的某个排列。

### 思路

排列意味着字符频次相同。

维护长度为 `len(s1)` 的窗口，比较窗口字符频次和 `s1` 是否一致。

### Python 代码

```python
from collections import Counter


def check_inclusion(s1: str, s2: str) -> bool:
    need_len = len(s1)
    if need_len > len(s2):
        return False

    need = Counter(s1)
    window = Counter()
    left = 0

    for right, ch in enumerate(s2):
        window[ch] += 1

        if right - left + 1 > need_len:
            left_ch = s2[left]
            window[left_ch] -= 1
            if window[left_ch] == 0:
                del window[left_ch]
            left += 1

        if window == need:
            return True

    return False
```

### 复杂度

- 时间复杂度：`O(n * k)`，`k` 是字符种类数。英文小写字母场景可视为 `O(n)`。
- 空间复杂度：`O(k)`

## 题 5：替换后的最长重复字符

### 题目

给定字符串 `s` 和整数 `k`，最多替换 `k` 个字符，使窗口内所有字符相同，返回最长长度。

### 思路

窗口长度为 `right - left + 1`。

如果窗口长度减去窗口内出现次数最多的字符数量 `max_count` 大于 `k`，说明需要替换的字符太多，必须收缩窗口。

### Python 代码

```python
from collections import defaultdict


def character_replacement(s: str, k: int) -> int:
    count = defaultdict(int)
    left = 0
    max_count = 0
    best = 0

    for right, ch in enumerate(s):
        count[ch] += 1
        max_count = max(max_count, count[ch])

        while (right - left + 1) - max_count > k:
            count[s[left]] -= 1
            left += 1

        best = max(best, right - left + 1)

    return best
```

### 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(k)`，`k` 是字符种类数

## 本模块总结

滑动窗口题目优先问：

```text
1. 题目是不是连续子串或连续子数组？
2. 窗口里要维护什么？和、计数、集合，还是最大频次？
3. 什么条件下窗口不合法？
4. 不合法时怎么移动 left？
5. 答案是在窗口合法时更新，还是窗口不合法前更新？
```
