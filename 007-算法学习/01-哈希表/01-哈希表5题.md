# 哈希表 5 题

## 学习过程

哈希表的核心价值是把“查找”从 `O(n)` 降到接近 `O(1)`。

常见模板：

```python
seen = {}

for i, x in enumerate(nums):
    if 需要的值 in seen:
        return ...
    seen[x] = i
```

适合场景：

- 查某个元素是否出现过。
- 统计元素出现次数。
- 找配对关系。
- 按某个特征分组。

---

## 题 1：两数之和

### 题目

给定数组 `nums` 和目标值 `target`，返回两个数的下标，使它们相加等于 `target`。

### 思路

遍历每个数 `num`，需要找的另一个数是 `target - num`。

用字典保存已经遍历过的数字和下标：

```text
key：数字
value：下标
```

如果 `target - num` 已经在字典里，说明找到答案。

### Python 代码

```python
from typing import List


def two_sum(nums: List[int], target: int) -> List[int]:
    seen = {}

    for i, num in enumerate(nums):
        need = target - num
        if need in seen:
            return [seen[need], i]
        seen[num] = i

    return []
```

### 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

## 题 2：有效的字母异位词

### 题目

给定两个字符串 `s` 和 `t`，判断 `t` 是否是 `s` 的字母异位词。

### 思路

字母异位词要求：

```text
字符种类相同
每个字符出现次数相同
```

用字典统计字符次数。

### Python 代码

```python
from collections import Counter


def is_anagram(s: str, t: str) -> bool:
    return Counter(s) == Counter(t)
```

不用 `Counter` 的写法：

```python
def is_anagram(s: str, t: str) -> bool:
    if len(s) != len(t):
        return False

    count = {}

    for ch in s:
        count[ch] = count.get(ch, 0) + 1

    for ch in t:
        if ch not in count:
            return False
        count[ch] -= 1
        if count[ch] < 0:
            return False

    return True
```

### 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(k)`，`k` 是字符种类数

## 题 3：数组中重复的数字

### 题目

给定一个整数数组，判断是否存在重复元素。

### 思路

用集合保存出现过的数字。遍历时如果数字已经在集合里，说明重复。

### Python 代码

```python
from typing import List


def contains_duplicate(nums: List[int]) -> bool:
    seen = set()

    for num in nums:
        if num in seen:
            return True
        seen.add(num)

    return False
```

更短写法：

```python
def contains_duplicate(nums: List[int]) -> bool:
    return len(nums) != len(set(nums))
```

### 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

## 题 4：最长连续序列

### 题目

给定未排序数组，找出最长连续数字序列的长度。

示例：

```text
nums = [100, 4, 200, 1, 3, 2]
最长连续序列是 [1, 2, 3, 4]
返回 4
```

### 思路

把所有数字放进集合。

只有当 `num - 1` 不在集合中时，`num` 才可能是一个连续序列的起点。

从起点开始不断找 `num + 1`。

### Python 代码

```python
from typing import List


def longest_consecutive(nums: List[int]) -> int:
    num_set = set(nums)
    best = 0

    for num in num_set:
        if num - 1 not in num_set:
            current = num
            length = 1

            while current + 1 in num_set:
                current += 1
                length += 1

            best = max(best, length)

    return best
```

### 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

关键点：不是每个数字都往后找，只有序列起点才开始找。

## 题 5：字母异位词分组

### 题目

给定字符串数组，把字母异位词分到同一组。

示例：

```text
["eat", "tea", "tan", "ate", "nat", "bat"]
返回：
[["eat","tea","ate"], ["tan","nat"], ["bat"]]
```

### 思路

字母异位词排序后结果相同：

```text
eat -> aet
tea -> aet
ate -> aet
```

用排序后的字符串作为 key，把原字符串放进列表。

### Python 代码

```python
from collections import defaultdict
from typing import List


def group_anagrams(strs: List[str]) -> List[List[str]]:
    groups = defaultdict(list)

    for word in strs:
        key = "".join(sorted(word))
        groups[key].append(word)

    return list(groups.values())
```

### 复杂度

假设有 `n` 个字符串，每个字符串平均长度为 `k`。

- 时间复杂度：`O(n * k log k)`
- 空间复杂度：`O(n * k)`

```text
3. 我是不是可以把某种特征作为 key？
4. 暴力双层循环能不能用 dict/set 优化？
```
