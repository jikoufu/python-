# 双指针 5 题

## 学习过程

双指针常见两种：

```text
左右指针：left 从左边走，right 从右边走。
快慢指针：fast 走得快，slow 走得慢。
```

适合场景：

- 有序数组。
- 回文判断。
- 原地删除。
- 反转字符串。
- 找链表中点、环。

---

## 题 1：有序数组的两数之和

### 题目

给定升序数组 `numbers` 和目标值 `target`，返回两个数的下标。

### 思路

左右指针：

- 如果两数之和等于 `target`，返回答案。
- 如果和太小，左指针右移。
- 如果和太大，右指针左移。

### Python 代码

```python
from typing import List


def two_sum_sorted(numbers: List[int], target: int) -> List[int]:
    left, right = 0, len(numbers) - 1

    while left < right:
        total = numbers[left] + numbers[right]

        if total == target:
            return [left, right]
        if total < target:
            left += 1
        else:
            right -= 1

    return []
```

### 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`

## 题 2：反转字符串

### 题目

给定字符数组，原地反转。

### 思路

左指针和右指针交换，然后向中间移动。

### Python 代码

```python
from typing import List


def reverse_string(s: List[str]) -> None:
    left, right = 0, len(s) - 1

    while left < right:
        s[left], s[right] = s[right], s[left]
        left += 1
        right -= 1
```

### 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`

## 题 3：验证回文串

### 题目

判断字符串是否是回文，只考虑字母和数字，忽略大小写。

### 思路

左右指针：

- 遇到非字母数字就跳过。
- 比较小写后的字符。
- 不相等则不是回文。

### Python 代码

```python
def is_palindrome(s: str) -> bool:
    left, right = 0, len(s) - 1

    while left < right:
        while left < right and not s[left].isalnum():
            left += 1
        while left < right and not s[right].isalnum():
            right -= 1

        if s[left].lower() != s[right].lower():
            return False

        left += 1
        right -= 1

    return True
```

### 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`

## 题 4：移动零

### 题目

给定数组，把所有 `0` 移动到末尾，同时保持非零元素相对顺序。

### 思路

用 `slow` 表示下一个非零元素应该放的位置。

遍历数组，如果遇到非零元素，就放到 `slow` 位置。

### Python 代码

```python
from typing import List


def move_zeroes(nums: List[int]) -> None:
    slow = 0

    for fast in range(len(nums)):
        if nums[fast] != 0:
            nums[slow], nums[fast] = nums[fast], nums[slow]
            slow += 1
```

### 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`

## 题 5：删除有序数组中的重复项

### 题目

给定升序数组，原地删除重复项，返回去重后的长度。

### 思路

`slow` 指向当前去重后数组的最后位置。

`fast` 向后扫描，如果遇到新值，就写到 `slow + 1`。

### Python 代码

```python
from typing import List


def remove_duplicates(nums: List[int]) -> int:
    if not nums:
        return 0

    slow = 0

    for fast in range(1, len(nums)):
        if nums[fast] != nums[slow]:
            slow += 1
            nums[slow] = nums[fast]

    return slow + 1
```

### 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`
