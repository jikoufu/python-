# 车载座舱经验

这个目录用于系统整理车载智能座舱相关经验，包括座舱基础、HMI 交互、语音、导航、多媒体、蓝牙、车控、车辆信号、车机通信、诊断日志、CANoe 工具、测试经验、项目经验和面试准备。

车载座舱经验适合和测试工程、AI 大模型测试结合起来学习。现在很多智能座舱已经接入语音助手、大模型助手、导航问答、车控 Agent、多模态交互，所以座舱测试不只是传统 UI 测试，也会涉及 AI、语音、车控安全和人机交互体验。
