# 12 CANoe 工具学习

CANoe 是车载测试领域非常常见的工具，常用于总线仿真、报文分析、ECU 仿真、诊断测试、自动化测试和车载网络问题定位。

在车载座舱测试中，CANoe 常用于：

- 查看 CAN 报文。
- 发送模拟车辆信号。
- 模拟车速、档位、车门、空调等状态。
- 验证车机 UI 和车辆信号是否一致。
- 做诊断请求和 DTC 相关测试。
- 配合自动化脚本做回归测试。
