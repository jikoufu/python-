# 02 MySQL

MySQL 是后端、测开、质量工程岗位最常被问到的数据库。学习重点不是背 SQL，而是能解释：表怎么设计、索引为什么生效或失效、事务为什么能保证一致性、慢查询怎么排查。

## 学完后你应该能做到

- 能启动本机 MySQL，并知道配置文件、数据目录、启动脚本在哪里。
- 能写基础 CRUD、分页、聚合、关联查询。
- 能设计常见业务表，知道主键、唯一索引、普通索引、联合索引怎么选。
- 能用 `EXPLAIN` 分析慢 SQL。
- 能解释事务、隔离级别、锁、MVCC。
- 能用 Python / SQLAlchemy 连接 MySQL。
